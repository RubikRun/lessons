6. while
- Repeats something, until a conditions stops being true
        while ( *condition* )
        {
            *piece of code*
        }
- Example: Go up from 0 to 10:
        int x = 0;
        while (x <= 10)
        {
            ...
            x++;
        }
- Example: Go down from 10 to 0:
        int x = 10;
        while (x >= 0)
        {
            ...
            x--;
        }


7. Functions
- A piece of code that you give a name to
        void *name*()
        {
            *piece of code*
        }
  Example:
        void sayHi()                             // The name is "sayHi"
        {
            cout << "Hi" << endl;
        }
- A function can be called by typing its name followed by ()
  Example:
        int main()
        {
            sayHi();                             // Calling sayHi 3 times
            sayHi();
            sayHi();
        }
- A function can return a result.
  There are different types of functions depending on what type of result they return.
        void  -  doesn't return anything
        int   -  returns an integer (whole number)
        bool  -  returns a boolean (true or false)
        char  -  returns a character
  Example:
        int f()                                // f is an int function
        {
            int x = 5;
            int y = 8;
            return x + y + 2;                  // 15 will be returned
        }
  When you call a function, it will be replaced by whatever it returns.
  Example:
            int main()
            {
                cout << f() << endl;
            }
      f is called, it returns 15, so "f()" is replaced by 15.
      Same as having this program instead:
            int main()
            {
                cout << 15 << endl;
            }
  Example:
            int main()
            {
                int x = f() - 9;
                cout << x << endl;
            }
      f is called, it returns 15, so "f()" is replaced by 15.
      Same as having this program instead:
            int main()
            {
                int x = 15 - 9;
                cout << x << endl;
            }


8. vector
- A list of multiple values    (called "elements")
- We must first include vector:
            #include <vector>
- Creating a vector:
            vector<int> v = { 3, 7, 9, 0, -3 };
- Index: a number that tells you where an element is in a vector
    (NOTE: indices start at 0)
            v[0]    <---    first element in a vector v
            v[2]    <---    third element in a vector v
- Adding an element to a vector:
            v.push_back(6)    <---    adding 6 to a vector v
- Vector's size is the number of its elements
            v.size()    <---    size of a vector v


9. for loop
            for (int i = 0; i < 100; i++)
            {
                ***some code***
            }
    Equivalent to a while loop:
            int i = 0;
            while (i < 100)
            {
                ***some code***
                i++;
            }
    Only difference is that the 3 parts of the loop
            int i = 0       <---    starting value
            i < 100         <---    condition - loop continues while true
            i++             <---    change value - happens after every iteration
    are all together in one line.
    Examples:
        Printing numbers from 5 to 25:
            for (int i = 5; i <= 25; i++)
            {
                cout << i << endl;
            }
        Calculating sum of numbers 3, 6, 9, 12, ..., 63, 66:
            int sum = 0;
            for (int i = 3; i <= 66; i += 3)
            {
                sum += i;
            }
            cout << sum << endl;
